# Time-Coach Skill Index

Welcome to time-coach, a calendar scheduling system for managing multiple priorities, projects, and habits while respecting immovable commitments.

---

## What This Skill Does

Time-coach runs two complementary modes:

- **MODE A (Sundays):** Weekly planning. Creates a skeleton week of calendar blocks distributed across your priorities, respecting buffers, windows, and catch-up needs.
- **MODE B (Mon–Sat):** Daily sync. Resolves emerging conflicts, detects at-risk activities, and adjusts for shortfalls.

Together, they keep your calendar conflict-free and aligned with your goals.

## Getting Started (New Users)

### Step 1: Copy System Docs
Everything in **90-system/** is yours as-is. No changes needed.

### Step 2: Set Up Your Config (5 minutes)
1. Copy `90-system/config.template.md` → `01-config/config.md`
2. Fill in your name, email, timezone, calendar IDs, activity codes, buffer rules
3. Create `01-config/objectives.md` with your weekly/monthly targets

### Step 3: Initialize Your Workspace
```bash
mkdir -p 02-exec/ 03-reports/
```

### Step 4: Run Pre-Flight
```bash
python scripts/validate_config.py 01-config/config.md
```

### Step 5: Run MODE A (Sunday)
Follow `90-system/runbook.md` § MODE A with your `01-config/` values. Result: a conflict-free week.

**Total time: ~15 minutes.**

## Key Concepts (5-Minute Overview)

**Priority Levels:** P0 (immovable: meals, routines) > P1 (must-do: work, education) > P2 (flexible: projects, habits)

**Two Modes:**
1. **MODE A (Sunday):** Distributes your weekly goals across the week in available time slots, respecting P0 immovable blocks and your preferred windows for each activity. Creates calendar blocks. Produces a weekly report.
2. **MODE B (Daily):** Detects conflicts (overlaps, buffer violations), resolves them by moving lower-priority blocks, flags at-risk activities (actual < goal), and calculates daily pacing needs. Updates calendar and log.

**Key Buffers:**
- Before/after exercise: 15 min before, 45 min after (recovery time)
- Post-meal transitions: 15 min (to avoid back-to-back eating/activity)
- Hard window: 08:00–22:30 (no calendar blocks outside this range)

**Catch-Up:** If you fell short last week, the allocation for this week automatically includes catch-up hours to get back on track.

## Questions?

- **Is time-coach for me?** Use it if you juggle 3+ projects, have inflexible meal/exercise times, and want to protect high-priority work from conflicts.
- **Can I customize it?** Yes. Edit `01-config/config.md` for your calendars, activities, and buffers. Edit `01-config/objectives.md` for your targets.
- **What if I'm not Gera?** All user-specific setup lives in `01-config/`. Replace with your own values. System docs in `90-system/` work for any user.
